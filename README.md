Ответ на вопрос 1
-
Что такое рекурсия и какие два обязательных компонента должна содержать рекурсивная функция?
-
Рекурсия представляет собой метод программирования, при котором функция вызывает саму себя для решения задачи путем разбиения ее на более простые подзадачи.

Два обязательных компонента рекурсивной функции:
- 

Базовый случай (base case) — условие завершения рекурсии, при котором функция возвращает результат без дальнейших рекурсивных вызовов. Базовый случай предотвращает бесконечную рекурсию и обеспечивает корректное завершение алгоритма.

Рекурсивный случай (recursive case) — часть функции, которая вызывает саму себя с измененными параметрами, приближающими решение к базовому случаю.


Функция генерации всех перестановок строки
-

Принцип работы:
-

Количество перестановок: Для строки длиной n количество всех возможных перестановок равно n! (n факториал)
permute(s, step + 1) 
Генерация перестановок: Алгоритм генерирует все возможные упорядочивания символов исходной строки

Лексикографический порядок: Во многих реализациях перестановки генерируются в лексикографическом порядке после предварительной сортировки s[step], s[i] = s[i], s[step]

Временная сложность: O(n!)

Объяснение сложности:
-

На первом шаге есть n возможных позиций для первого символа

На втором шаге остаётся (n-1) возможных позиций для следующего символа

На третьем шаге — (n-2) позиций, и так далее

Общее количество вариантов: n × (n-1) × (n-2) × ... × 2 × 1 = n!
