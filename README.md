Ответ на вопрос 1
-
Что такое рекурсия и какие два обязательных компонента должна содержать рекурсивная функция?
-
Рекурсия представляет собой метод программирования, при котором функция вызывает саму себя для решения задачи путем разбиения ее на более простые подзадачи.

Два обязательных компонента рекурсивной функции:

Базовый случай (base case) — условие завершения рекурсии, при котором функция возвращает результат без дальнейших рекурсивных вызовов. Базовый случай предотвращает бесконечную рекурсию и обеспечивает корректное завершение алгоритма.

Рекурсивный случай (recursive case) — часть функции, которая вызывает саму себя с измененными параметрами, приближающими решение к базовому случаю.
